# YuGiOh-文件名修改

#### 介绍
steam玩YuGiOh，如果有大量自抽号，需要频繁切换账号的话，可以使用该脚本一键修改目录名称，实现快速切号


#### 使用说明

1.  打开软件，修改对应的游戏目录，到LocalData即可 
![输入图片说明](image.png)
2.  打开目录下的YuGiOh.txt，填入你的卡组+游戏文件夹代码，参照原本就有的
3.  打开软件就可以用了
